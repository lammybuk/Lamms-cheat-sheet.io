Lamms' cheat sheet

Here are some common html tags for setting up a basic UI.

Tag Description
<html> The wrapper element for all html tables.
<header> A header for a document or a section
<nav> Set of navigation links
<section> Section in a document
<article> Independent, self-contained content
<aside> Content aside from the content (like a sidebar)
<footer> A footer for a document or a section
<details> A additional details that the user can open and close on demand
<summary> A heading for the details element
